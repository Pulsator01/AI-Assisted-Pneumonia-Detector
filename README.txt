# AI-Assisted-Pneumonia-Detection
This model can detect pneumonia based on X-Ray images of your lungs..
Required Prerequisites - Installation of the necessary modules mentioned in the code. 
                          Python 3.10
 Dataset used for training the model can be downloaded from the link given below
 https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia
 IMPORTANT: ADD THE PATH WITHOUT QUOTES
 To open pneumonia_classification.py in streamlit,
type streamlit run pneumonia_classification.py in the python terminal
 Make sure to change the directory of all the paths to the data/ modules to a suitable directory in your system.
